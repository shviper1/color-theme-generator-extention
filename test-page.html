How to Test

  1. Open the extension popup by clicking the icon in your Chrome toolbar
  2. Try preset themes in the Presets tab - click any theme card
  3. Create custom themes in the Custom tab - use the color pickers
  4. Use color harmony - click Complementary, Analogous, or Triadic buttons
  5. Extract colors from this page in the Extract tab
  6. Try AI features - select a mood or describe your ideal theme
  7. Check accessibility - use the Auto-Fix Contrast feature
  8. Highlight elements - see which page elements are affected
