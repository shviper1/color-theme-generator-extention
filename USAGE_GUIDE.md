## Custom Theme Creator

### Creating Your First Custom Theme

#### Step 1: Choose Base Colors

1. Go to the **Custom** tab
2. Click the color picker for **Background**
3. Choose your base background color
4. Select a **Text** color that contrasts well
5. Pick **Primary**, **Secondary**, and **Accent** colors

#### Step 2: Use Color Harmony

Instead of manually picking each color, use the Harmony Generator:

- **Complementary**: Creates opposite colors (great for contrast)
  - Example: Blue ‚Üí Orange
  
- **Analogous**: Creates adjacent colors (smooth, harmonious)
  - Example: Blue ‚Üí Blue-Green ‚Üí Green
  
- **Triadic**: Creates 3 balanced colors
  - Example: Red ‚Üí Yellow ‚Üí Blue
  
- **Tetradic**: Creates 4 balanced colors
  - Perfect for complex designs
  
- **Monochromatic**: Creates variations of one color
  - Great for cohesive, professional looks

**How to Use:**
1. Set your **Primary** color
2. Click a harmony button (e.g., "Complementary")
3. Watch the other colors update automatically!

#### Step 3: Check Accessibility

The **Contrast Badge** next to the text color shows:
- üü¢ **AAA** - Perfect! Exceeds all standards
- üü° **AA** - Good! Meets standard requirements
- üî¥ **FAIL** - Needs improvement

**Target:** Aim for at least **AA** level for text.

#### Step 4: Preview

The **Preview** section shows:
- How headings will look
- Sample paragraph text
- Button appearance
- Link colors

Make sure everything is readable!

#### Step 5: Apply & Save

1. Click **Apply Custom Theme** to test on the current page
2. Click **Save Custom Theme** to remember it for this website
3. Use **Export** to share with others or back up

## Tips & Best Practices

### Accessibility Best Practices

1. **Always Check Contrast**
   - Use the contrast badge
   - Aim for AA or better
   - Test with different text sizes

2. **Consider Color Blindness**
   - Don't rely solely on color
   - Use text labels
   - Test with high contrast

3. **Use the Auto-Fix**
   - Let AI optimize your theme
   - Review the suggestions
   - Make final adjustments

### Design Best Practices

1. **Start with Presets**
   - Learn from well-designed themes
   - Understand color relationships
   - Build from there

2. **Use Harmony Tools**
   - Don't pick colors randomly
   - Use complementary for contrast
   - Use analogous for harmony

3. **Less is More**
   - Limit to 2-3 main colors
   - Use accent colors sparingly
   - Keep it simple

4. **Test on Real Content**
   - Apply to actual websites
   - See how it looks with images
   - Check different page types

### Performance Tips

1. **Save Your Favorites**
   - Use the Save button
   - Themes load automatically
   - No need to reapply

2. **Export Important Themes**
   - Back up custom themes
   - Share with team members
   - Keep a library

3. **Reset When Needed**
   - Quick reset button available
   - No permanent changes to sites
   - Experiment freely

## Troubleshooting

### Theme Not Applying

**Problem**: Clicked apply but nothing changed

**Solutions:**
1. Make sure you're on an `http://` or `https://` page
2. Refresh the page after applying
3. Check if the site has very specific CSS that overrides
4. Try the "Highlight Elements" feature to see what's affected

### Contrast Badge Shows FAIL

**Problem**: Can't get good contrast

**Solutions:**
1. Use the **Auto-Fix Contrast** feature
2. Make text much lighter or darker
3. Change background color
4. Try the complementary harmony

### Colors Look Different Than Expected

**Problem**: Preview looks good but page doesn't

**Solutions:**
1. Some websites have very specific styles
2. Try the "Highlight Elements" to see what's targeted
3. Some elements may have inline styles
4. Refresh the page

### Extension Not Loading

**Problem**: Icon doesn't appear or doesn't work

**Solutions:**
1. Reload the extension in `chrome://extensions/`
2. Check that it's enabled
3. Try restarting Chrome
4. Make sure it's the latest version

### Custom Theme Not Saving

**Problem**: Theme doesn't persist

**Solutions:**
1. Make sure you clicked "Save Custom Theme"
2. Check Chrome storage permissions
3. Try exporting and re-importing
4. Clear extension storage and recreate
